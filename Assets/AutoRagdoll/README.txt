This little tool helps you automatically create ragdoll for all humanoid models

Open it in Menu "Tool/AutoRagdoll" or press Alt+R

Drag and drop any humanoid model from the Hierarchy to create it

its aim to speed the character creation and avoid to drap and drop so many bones to create ragdoll

If you want to create NPC like use plugin Realistic FPS Prefab
https://www.assetstore.unity3d.com/en/#!/content/7739
it will boost your developement
